## 🚀 Introduction
AI Trading Bot is an advanced AI-driven trading bot that analyzes live Twitter data to make smart trading decisions. By utilizing the Twitter API for real-time tweet fetching and the Aptos Blockchain for transaction execution, our bot autonomously trades assets based on sentiment analysis.

---

## 📌 Features
- **Real-time Twitter Sentiment Analysis**: Analyzes tweets from influencers like @ElonMusk and @APompliano.
- **Automatic Trading**: Executes buy/sell orders based on AI-driven sentiment analysis.
- **Blockchain-Based Transactions**: Uses Aptos smart contracts for secure and transparent trading.
- **24/7 Autonomous Trading**: Runs continuously without human intervention.
- **Portfolio Management**: Tracks profits and manages asset holdings in real-time.

---

## 🔍 How It Works
<img src="https://github.com/user-attachments/assets/88ae2d29-4052-4ccd-97fa-c93e15658f21" width=800 height =600></img>

### Step 1: Twitter Sentiment Analysis
- Fetch real-time tweets from influencers using Twitter API (e.g., @ElonMusk, @APompliano).
- Preprocess tweets: remove stop words, hashtags, and links.
- Apply sentiment classification using Move AI (bullish = buy, bearish = sell).
- Extract tweet timestamps, text, and metadata in real-time.

### Step 2: AI-Based Trading Decisions
- Gather historical price data of Aptos-based assets.
- Use AI to analyze and predict market trends based on sentiment patterns.
- Combine AI sentiment analysis with real-time market trends.

### Step 3: Aptos Smart Contract Execution
- Use Aptos TypeScript SDK to interact with on-chain assets.
- Execute buy/sell transactions directly on the Aptos blockchain.
- Manage the bot’s portfolio autonomously through smart contracts.

## 🚀 Getting Started
### 1. Clone the Repository
```sh
git clone https://github.com/yourusername/ai-trading-bot.git
cd ai-trading-bot
```

### 2. Install Dependencies
```sh
npm install
```

### 3. Set Up API Keys
Create a `.env` file and add the following credentials:
```plaintext
TWITTER_API_KEY=your_twitter_api_key
TWITTER_API_SECRET=your_twitter_api_secret
APTOS_API_KEY=your_aptos_api_key
PRIVATE_KEY=your_private_key
```

### 4. Run the Bot
```sh
npm start
```
